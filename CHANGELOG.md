# Changelog

## 0.1.0 (2026-03-19)

- Initial release
- Parse .env files with support for quoted values, comments, and multiline strings
- Variable interpolation: `${VAR_NAME}` syntax
- Multi-file layering with priority ordering
- Type-safe accessors: get_as, get_bool, get_list
- Required variable validation
- DotEnv struct for non-global usage
